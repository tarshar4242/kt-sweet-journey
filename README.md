## 一、這是什麼

這不是一份「看的」行程表，而是一份**能跟著執行、邊走邊打勾的自由行導引手冊**。

專為 **K（先生）與 T（太太）** 兩人的 2026 年夏季日本之旅打造，從台灣出門到平安回家，每一段交通、每一個機場流程都能逐項確認，把「第一次自由行的混亂與壓力」降到最低。

> 💡 **設計初衷**：第一次自由行最怕的不是迷路，是「不知道現在該在哪、下一班幾點、來不及怎麼辦」。這份手冊把每一步都攤開來，讓壓力點看得見、可掌握。

## 二、核心功能

| 功能 | 說明 |

### 🚦 班次風險標示（最重要）

| 風險 | 標示 | 代表路線 | 班次間隔 |
|------|------|---------|---------|
| 🔴 高 | 錯過等久，要卡準 | N'EX、海賊船、Skyliner | 30–60 分一班 |
| 🟡 中 | 注意時間 | 箱根登山電車、登山巴士 | 約 15–20 分一班 |
| 🟢 低 | 班次密集放心 | 山手線、JR 各線、纜車 | 約 3–10 分一班 |

> ⚠️ **防延誤三原則**：紅標三段班次最稀疏，抓時間務必預留一班緩衝；藍綠色 JR 與山手線班次密集，臨時錯過也不怕；實際時刻出發前用 Google Maps 再對一次最準。

## 六、使用方式

```
1. 用手機瀏覽器打開網址
2. 跟著行程走，每完成一項就點一下打勾
3. 頂端進度條會即時更新完成度
4. 紅標交通段落 = 班次稀疏，務必提前到月台
5. 出發前可按「重置進度」清空重來
```

**線上網址**：`https://tarshar4242.github.io/kt-sweet-journey/kt-sweet-journey.html`

**同行工具・隨身口譯機**：`https://tarshar4242.github.io/kt-sweet-journey/interpreter.html`
中⇄日／中⇄英雙向即時翻譯——自動聽出誰講哪種語言：上區顯示對方的話＋中文翻譯，下區顯示你的話＋外語翻譯，可存逐字稿。首次使用需在 ⚙️ 貼上 OpenAI API 金鑰（金鑰只存在自己手機的瀏覽器裡）。
