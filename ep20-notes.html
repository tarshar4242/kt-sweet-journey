CHAPTER 01

1. 基礎設置與使用策略

在正式使用 Claude 之前，先理解其運作機制與額度限制，是發揮效能與控管成本的第一步。

額度與成本管理

  • Claude 提供免費版與付費版（Pro）兩種方案。
  • 免費版：約可進行 50 次對話。
  • Pro 版：約可進行 175–200 次對話。
⚠️ 重要提醒：在「設定（Settings）」中，建議關閉「額外增量付費」相關選項。除非是企業或專案需求，否則應避免因高頻使用而產生非預期費用。

介面語言與操作建議

  • 目前 Claude 的操作介面仍以英文為主。
  • 雖然可透過瀏覽器即時翻譯為中文，但在執行較複雜的指令時（例如資料匯出、程式碼輸出、檔案處理），建議切換回原始英文介面，以降低錯誤發生的機率。
CHAPTER 02

2. 隱私與個人化配置

良好的隱私設定，能同時保護資料安全，也能讓 AI 更貼近你的工作習慣。

隱私保護設定

  • 建議關閉：改善模型（Help improve Claude）——用來訓練和改進 Anthropic AI 模型的選項。
  • 建議關閉：分享位置（Location metadata / Share location）。

如此可避免對話內容被用於模型訓練，並降低個人與地理資訊外洩的風險。

🧠 簡報另提到 Memory（記憶）功能：允許 Claude 搜尋過去聊天中的相關詳情，從聊天歷史生成記憶，記住你聊天中的相關背景——是否開啟同樣屬於隱私取捨的一環，建議依個人需求評估後再決定。

自定義指令（使用者偏好 / Personal preferences）

你可以在此設定 Claude 的長期行為規則，例如：

  • 指定角色定位（如：資料分析師、政策研究員、行銷顧問）。
  • 規定回覆風格（條列式、學術寫作、政策分析語氣）。
  • 指定語言或呈現方式（例如固定使用繁體中文、或特定格式）。

這一區塊等同於為 Claude 建立「長期工作人格設定」。簡報中對應的欄位是「Personal preferences」，題目寫著 "What personal preferences should Claude consider in responses?"，範例填法如 "Be concise, use bullet points, always provide examples."——這段設定會變成帳號級預設，跨對話通用，可套用到教學／顧問型、簡報型、SOP／流程型等不同任務。

MCP 擴充連結（外部工具）

Claude 可連結 Google Drive、日曆與 Gmail 等服務，但目前功能以「讀取」為主，尚未支援直接寫入或自動修改內容。在實務上，較適合作為輔助判讀與整理工具，而非自動化執行工具。

CHAPTER 03

3. 深度思考與寫作風格

Claude 的核心強項，在於結構清楚、推理完整、文字表現穩定。

深度思考模式（Extended thinking）

點擊介面中的時鐘圖示，即可啟用「深度思考」功能。此模式下，模型會花更多時間進行邏輯推演，適合用於：

  • 技術或政策方案比較。
  • 多方案利弊分析。
  • 高風險或高影響決策判斷。

簡報進一步把「在難題上多想一步、降低失誤」的流程拆成四步：

  1. 定義問題（Define）：出發解讀問題核心。
  2. 探索分支（Explore）：探索不同解法分支。
  3. 深度分析（Analyze）：針對各分支做深度分析。
  4. 綜合方案（Synthesize）：綜合各分析結果成一個方案。

寫作風格客製化（Writing style）

Claude 內建多種寫作風格，簡報列出四種：

風格特性與適用場景
Normal一般溝通、日常任務的標準語氣。
Concise重點直接、精簡有力，適合快速摘要、條列、標題。
Formal正式、專業，用詞較進階，適合正式文件、商業書信、報告、郵件、學術寫作。
Explanatory逐步拆解概念，包含範例，聚焦「如何運作」與「為什麼」，適合教學、指南、教育內容。
用「風格」控制呈現方式，用「prompt」控制任務內容。
—— 簡報 Writing style 一節

更進階的作法是打造專屬寫作風格：

  1. 提供一段你自己的既有文字。
  2. 請 Claude 分析你的語氣與結構特徵（例如：條列導向、台灣政策語境、行政體系寫作）。
  3. 將其作為「專屬寫作風格」長期套用。

這一步驟，對於研究者、公部門寫作與課程講師特別關鍵。

CHAPTER 04

4. Claude Artifacts：視覺化與 SVG 生成

Artifacts 是 Claude 在實務應用上最具代表性的功能之一——把輸出變成「作品」與「可分享的成品」。

SVG 圖形生成

可直接要求 Claude 產出：

  • 文字雲。
  • 圓餅圖。
  • 架構流程圖。
  • 簡報用示意圖。

並以 SVG 向量格式輸出。

簡報實戰應用

將 SVG 檔案下載後拖入 PowerPoint，點選「轉換為圖形」，即可：

  • 拆解為可編輯物件。
  • 自由更換顏色與字型。
  • 作為正式簡報素材使用。

這對於政策簡報、教學投影片與報告視覺化特別實用。

遊戲與小工具示範

Claude 亦可生成簡單的小型互動工具（如貪食蛇、俄羅斯方塊），適合用於：

  • 教學展示。
  • 技術概念說明。
  • AI 能力示範。

Artifacts 五步驟（簡報補充）

簡報把 Artifacts 的使用流程整理成五步，特別適合圖、表、流程圖、互動小工具、可重混的模板：

  1. 創建（Create）
  2. 編輯（Edit）
  3. 預覽（Preview）
  4. 分享（Share）
  5. 協作（Collaborate）
CHAPTER 05

5. Connectors：把 Claude 連到你的資料與工具

簡報用一張「Claude 功能地圖」把整個網頁介面分成回應層、工作空間層、外部整合層來導覽，Connectors 就落在「外部整合層」。連接不等於永久儲存；權限與資料邊界要先講清楚。

新增自訂連接器

可以新增自訂連接器，讓 Claude 連到自己或公司內部的系統／服務，在設定中運作及回應記憶。伺服器需要 OAuth 驗證，重複利用專屬功能按鈕。簡報列出的常見 Connectors 包括：

  • Google Drive
  • Gmail
  • GitHub
  • Google Calendar
把 Claude 接上你的資料和工具（Connect Claude to your data and tools）。
—— 簡報 Connectors 一節標題

連接流程與權限

  1. 選擇服務。
  2. 授權連接。
  3. 設置權限。
  4. 確認範圍。

多數服務預設為「只讀」權限——這也呼應課程筆記中「MCP 擴充連結目前以讀取為主，尚未支援直接寫入」的說法。

自訂 Connector 詳細設定（Custom Connector Setup）

簡報中「Add custom connector」的設定畫面需要填：

  • Name：輸入 connector 名稱。
  • Remote MCP server URL：例如 https://api.mcp-server.com。
  • Advanced settings：OAuth／安全性配置。

設定流程五步：

  1. 開始設定新 Connector。
  2. 輸入基本資訊。
  3. 配置伺服器 URL。
  4. 設定進階安全選項。
  5. 儲存並驗證連線。
⚠️ 風險提醒（簡報明確列出）：未經驗證的伺服器、敏感資料傳輸風險、安全設定不足——新增自訂連接器前務必留意。
CHAPTER 06

6. Projects 專案功能與 Research 研究功能

Projects：把聊天變成「可管理的工作空間」

一個 Project = 一套任務、資料、規則與產出格式。設定步驟：

  1. 設定專案名稱與目標。
  2. 匯入或創建教材與規範資料。
  3. 定義固定輸出格式與任務清單。
  4. 配置專案規則與工作流。

Research：讓 Claude 做「可查證的網路研究」

從「我覺得」變成「我查到，且可引用」：

  • Research OFF：只能憑內部知識回答。
  • Research ON：會實際搜尋官方文件、大型媒體、部落格／論壇等來源，並附引用。
CHAPTER 07

7. 多端工具與瀏覽器擴充

桌面版應用

桌面版 Claude 適合：文字工作者、工程與資料分析人員。其優點在於同步速度快，且與本機環境整合度高。

簡報補充的安裝流程（Mac/Windows 類似）：

  1. 下載應用程式（Download App）。
  2. 執行安裝檔（Run Installer）。
  3. 登入帳戶（Log In）。
💡 桌面端＝更貼近本機工作流（尤其是 Claude Code／MCP）。

Chrome 瀏覽器擴充功能

Claude 的瀏覽器擴充功能可直接讀取當前網頁內容，操作方式接近即時助理。

實務應用範例：在 Google 地圖搜尋餐廳時，開啟側邊欄，要求 Claude：

實例「整理目前畫面中所有餐廳名稱、評價與距離，並製成表格。」

此類操作特別適合資訊蒐集與快速整理。安裝連結：Claude in Chrome（Chrome 線上應用程式商店）。

CHAPTER 08

8. Excel 與數據分析應用

Excel 整合

可透過增益集或直接貼上資料，要求 Claude：

  • 生成 Excel 公式。
  • 建立樞紐分析表。
  • 視覺化趨勢圖。

在某些邏輯判斷與資料理解上，Claude 的回應往往較直覺、可解釋。安裝連結：Claude in Excel（Microsoft AppSource）。

學術與統計分析

Claude 可協助撰寫 R 或 Python 程式碼，用於：

  • 成對樣本或獨立樣本 t 檢定。
  • 變異數分析（ANOVA）。
  • 迴歸分析。
✅ 實務小技巧：在學術投稿前，可請 Claude 協助檢查：P 值是否達顯著水準、論述是否符合學術書寫慣例、統計結果敘述是否一致。

10. 重點整理：快速回顧 EP20

  • 先搞懂額度（免費 50 次／Pro 175–200 次）並關閉額外增量付費，複雜指令切回英文介面。
  • 隱私設定建議關閉改善模型與分享位置；用 Personal preferences 建立帳號級長期人格設定。
  • 深度思考（時鐘圖示，定義→探索→分析→綜合）適合高風險決策；寫作風格四選一（Normal／Concise／Formal／Explanatory），也能訓練專屬風格。
  • Artifacts／SVG 能產出文字雲、流程圖、簡報素材，甚至小遊戲，五步驟：創建→編輯→預覽→分享→協作。
  • Connectors 把 Claude 接上 Google Drive／Gmail／GitHub／Calendar，多為只讀；自訂 Connector 需注意 OAuth 與伺服器安全風險。
  • Projects 把聊天變工作空間；Research 把「我覺得」變「我查到，且可引用」。
  • 桌面版同步快、整合本機環境；Chrome 擴充能即時整理當前網頁內容為表格。
  • Excel 整合可做公式／樞紐分析／趨勢圖；統計分析可協助 t 檢定、ANOVA、迴歸與投稿前檢查。
  • 官方提供多款 Skills（web-artifacts-builder、skill-creator、theme-factory 等），依生活、行銷、設計、法律、人資、財務、數據科學、工程、營運、產品、銷售等場景分類。
Claude 不只是聊天工具。
透過個人化設定、深度推理、SVG 視覺化與 Excel 整合，它能成為一個橫跨資料處理、分析思考與簡報製作的整合型工作夥伴。—— 課程筆記結語
